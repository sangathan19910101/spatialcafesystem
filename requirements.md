1. Dynamic Cafe Layout Designer
Manager can:
	create floors
	define grid size (rows/columns)
	drag/place tables
	resize tables
	merge table spaces
	assign table IDs/names
	set seating capacity
	
2. Live Table Status System
Tables can be:
	Available
	Reserved
	Occupied
	Cleaning
	Disabled
Visual colors:
	Green = free
	Red = occupied
	Yellow = reserved
	

3. Order Management
When clicking a table:
	open order panel
	add food items
	adjust quantity
	special notes
	remove items
	split bill (optional advanced feature)
	
4. Billing System
Auto-generate:
	subtotal
	tax/VAT
	discounts
	grand total
Optional:
	printable receipt
	PDF bill
	QR payment

6. Analytics Dashboard
Add:
	busiest tables
	daily revenue
	most ordered food
	peak hours
